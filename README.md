# 🐺🐰 LUPUS & LEPUS (狼兔杀)

> **“在可爱的兔窝中，恶狼的阴影正在蔓延...”**

**LUPUS & LEPUS (狼兔杀)** 是一款基于 Web 的多人在线社交推理卡牌游戏。玩家将扮演兔子、恶狼或中立角色，通过推断身份、投票和执行任务来达成阵营目标。

本项目采用**纯前端 P2P (Peer-to-Peer) 架构**，无需任何后端服务器，利用 WebRTC 技术实现玩家直连。这意味着游戏极其快速、隐私安全且**永久免费**。

🔗 **在线试玩**: [点击这里进入游戏](https://lupus-and-lepus-game.pages.dev/)

## ✨ 核心特性

### 🎮 游戏功能亮点

* **全新阵营系统**:
    * **🐰 兔子阵营**：目标是通过成功执行任务来保护社区。
    * **🐺 恶狼阵营**：目标是潜伏在好人中，通过破坏任务来赢得胜利。
    * **⚖️ 中立阵营**：拥有独特的胜利条件，例如囤积手牌、完成特定行动等。
* **多样化角色技能**：包含 **哨兵兔** (探路)、**伪装狼** (光环伪装)、**和平鸽** (查验阵营) 等多种主动/被动技能，极大地提升了游戏深度和推理性。
* **自动化流程**：自动发牌、自动计票、自动结算胜负，确保流畅的游戏体验。
* **便捷开局**：支持一键添加 AI 机器人填补空位，无需繁琐操作。

### 🛠️ 技术亮点

* **去中心化架构 (Serverless P2P)**：使用 **PeerJS** 实现浏览器点对点通信。房主即服务器，数据存储在本地。
* **网络自动愈合**：内置断线重连机制，网络波动或切换后台后会自动尝试重连并同步最新战局。
* **全球/大陆网络优化**：混合配置 Google、Twilio 以及国内极速 STUN 服务器 (腾讯/小米)，确保国内外玩家均能秒进房间。
